## Dataset
Os dados utilizados neste projeto são públicos e foram obtidos no Kaggle.

O dataset contém informações de um e-commerce brasileiro, incluindo:
- Pedidos
- Itens dos pedidos
- Produtos
- Clientes

Os dados passaram por etapas de limpeza, transformação e integração
antes da análise.# projeto_analise_de_dados
Esse é um projeto para mostrar minhas habilidades como analista de dados usnado Python e outras tecnologias

## Como obter os dados

Os dados utilizados neste projeto são públicos e não estão versionados
neste repositório.

Para reproduzir a análise:

1. Acesse o Kaggle
2. Baixe o dataset **Brazilian E-Commerce Public Dataset (Olist)**
3. Extraia os arquivos CSV
4. Copie os arquivos abaixo para a pasta `data/raw/`:
   - orders.csv
   - order_items.csv
   - products.csv
   - customers.csv
